# Overview

ThePhoenixEngine is the core AI production platform developed under The Phoenix Studios.

It is designed as a modular enterprise system capable of powering multiple intelligent engines through one common infrastructure.

Current modules include Humaniser, AI Detection, Batch Processing and Report Generation.

Future Phoenix Engines will reuse the same production core.
